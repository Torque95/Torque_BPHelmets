 GTA Helmet Armor Fix

This lightweight FiveM client-side script ensures that wearing base-game GTA helmets does **not** provide unintended bulletproof (headshot protection) effects.

 What It Does

Some helmets in vanilla GTA (like bulletproof or military helmets) silently prevent headshot kills by disabling critical hits. This script re-enables vulnerability to headshots **only when a helmet is equipped**, ensuring fair combat without removing or blocking helmet use.

✅ Allows all helmets to be worn  
✅ Keeps player armor (vests, inventory, etc.) untouched  
✅ Removes passive "helmet armor" from Rockstar’s base-game helmets  

 Installation

1. Drop the folder (`Torque_BPHelmets`) into your FiveM resources directory.
2. Add this line to your `server.cfg`:

ensure Torque_BPHelmets
